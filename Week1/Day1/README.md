# Development Environment Setup

This document outlines the basic setup steps completed before starting Python development.

## Prerequisites Completed

### 1. Installed Python

Verified Python installation on the local machine.

```bash
python --version
```

---

### 2. Created a Python Virtual Environment

Created a virtual environment named `venv`.

```bash
python -m venv venv
```

Activated the virtual environment:

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

---

### 3. Created a GitHub Account

Created a GitHub account for source code management and version control.

GitHub Website:

https://github.com/sudhachandiranna-hub/GENAI-Program.git

---

### 4. Installed Visual Studio Code (VS Code)

Downloaded and installed Visual Studio Code.

VS Code Website:

https://code.visualstudio.com

Verified installation:

```bash
code --version
```

---

### 5. Signed In to GitHub from VS Code

Connected VS Code to GitHub using the built-in GitHub authentication.

Steps:

1. Open VS Code.
2. Click **Accounts** (bottom-left corner).
3. Select **Sign in with GitHub**.
4. Complete browser authentication.

---

### 6. Installed GitHub Copilot Extension

Installed the following VS Code extensions:

* GitHub Copilot
* GitHub Copilot Chat

Steps:

1. Open Extensions (`Ctrl + Shift + X`).
2. Search for **GitHub Copilot**.
3. Click **Install**.
4. Search for **GitHub Copilot Chat**.
5. Click **Install**.

---

### 7. Activated GitHub Copilot Chat

After installation:

1. Sign in with GitHub.
2. Open the Copilot Chat panel.
3. Verify Copilot Chat is available.
4. Start asking coding questions directly from VS Code.

---
